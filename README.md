### 👨‍💻 About Me

I am currently in my third year at the **Universidad José Cecilio del Valle (UJCV)**. My technical approach is versatile: I bridge the gap between pure software development, experimenting with Artificial Intelligence models in local environments, and hardware/electronics engineering.

* 🚀 **Full-Stack & Mobile Development:** I build dynamic web applications and cross-platform mobile solutions using React Native (Expo), Next.js, and Java (Swing/MVC), powered by robust backends like Supabase and SQL.
* 🤖 **Local AI Enthusiast:** I optimize workflows by setting up and configuring local agents and LLMs (Ollama, DeepSeek, Qwen) on my machine, leveraging tools like Continue and Cline for development.
* 🔌 **Hardware & Electronics:** I design and implement physical circuits and laboratory projects, including hands-on work like analog-style traffic light simulations using Arduino Uno and ESP8266 microcontrollers.
* ⚡ **Fun fact:** When I'm away from the keyboard, I split my time between doing mechanical maintenance on my vehicles (Honda CR-V and Yamaha YBR), gaming (EA FC 26, Resident Evil Village, GTA), and hanging out with my cats (Fermina, Felix, and Gato).
